# Synopsis

Haskell implementation of coalgebraic logic programming. Experimental,
development version.


# Installation


## Standard installation using `cabal`

Run from the project directory:

> cabal install


## Installation with optional tests

Run from the project directory:

> cabal install --enable-tests
